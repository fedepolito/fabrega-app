# Tienda Fabrega - React Native

Aplicación móvil de e-commerce desarrollada con React Native y Expo.

## Tecnologías

- React Native
- Expo
- TypeScript
- Expo Router
- Tailwind CSS

## Cómo ejecutar

1. Clonar el repositorio:
```bash
git clone https://github.com/fedepolito/fabrega-app.git

2. Instalar dependencias:
```bash
npm install

3. Iniciar servidor:
```bash
npm start

Nota
Este es un proyecto de aplicación móvil. Para verlo completamente, necesitas:
* App Expo Go en tu celular (iOS/Android)
* O presionar w en la terminal para ver la versión web
